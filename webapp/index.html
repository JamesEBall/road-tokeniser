About

This visualisation is the Phase A deliverable for the AI for Safer Roads Innovation Challenge. It identifies road segments where the posted speed limit is misaligned with the Safe System recommended speed, given the road's geometry, vulnerable road user exposure, and crash history.

Each segment is a 5–35 m token derived from OpenStreetMap road geometry, enriched with VRU proximity features and severity-weighted crash counts. The Safe System speed is derived from a rule cascade (motorway pass-through, VRU exposure, junction proximity, curvature, road class).

Source: github.com/JamesEBall/road-tokeniser
